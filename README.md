## About catalog-app

Example of simple products list with filters

 - use sail, containers with redis, laravel app, pgsql
 - <a href="/api/documentation">Swagger api</a>

## Start instructions:
- copy .env.example to .env
- install and up sail
- run artisan migrations and seeding - php artisan migrate:fresh --seed
- done!
